🍽️ Gestión Restaurante · Ver. 2.1.0

Sistema completo de gestión de restaurante. Controla zonas de atención, pedidos, cuentas, usuarios, reportes y más.

Desarrollado por: Andrey Acuña

⚙️ Tecnologías Utilizadas

  • Node.js + Express (backend)
  • SQLite3 (base de datos)
  • JavaScript Vanilla (frontend)
  • HTML/CSS
  • bcryptjs, express-session, multer, nodemon, etc.

📂 Estructura del Proyecto

restaurant-app/
├── server/              → Lógica del backend
│   ├── app.js           → Entrada del servidor
├── public/              → Interfaz del sistema
├── test_data.sql        → Datos de prueba
├── package.json         → Dependencias

🧠 Lógica del Sistema

  • Control de sesión única y usuarios con permisos personalizados.
  • Zonas: Mesas (salón) y Bancos (barra), visibles según rol.
  • Pedidos: Agregar productos, editar, pagar por separado o completo.
  • Pagos: Efectivo, tarjeta y crédito con validación de administrador.
  • Configuraciones: idioma, moneda, tipo de cambio, impresora, fecha.

▶️ Instrucciones para Desarrolladores

npm install
npm run dev      # Modo desarrollo
npm start        # Producción

Cargar datos de prueba:

sqlite3 database.db < test_data.sql
